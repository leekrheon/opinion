이용약관 및 개인정보 처리방침
Opinion by IMBY · 최종 수정 2026년 4월

서비스를 이용하기 전 본 약관을 반드시 읽어주세요. 서비스에 최초 접속하거나 계속 이용하는 경우, 아래 약관 및 개인정보 처리방침에 동의한 것으로 간주됩니다. 동의하지 않으실 경우 즉시 이용을 중단해 주세요.

제1조 서비스 개요
Opinion(이하 '서비스')은 YouTube 영상의 댓글을 수집·분석하여 여론 동향을 시각화하는 웹 서비스입니다. 운영자는 Opinion by IMBY이며, 문의는 support@inmybackyard.kr로 가능합니다. 서비스는 YouTube Data API v3를 통해 공개된 댓글 데이터를 조회하며, 비공개·삭제된 콘텐츠에는 접근하지 않습니다.
제2조 이용 자격 및 방법
  • 만 14세 이상이면 누구나 이용할 수 있습니다. 만 14세 미만은 서비스를 이용할 수 없습니다.
  • 게스트 모드는 1일 6회 무료 분석을 제공하며, 자정(00:00 KST)에 자동 초기화됩니다.
  • 자신의 YouTube Data API KEY를 등록하면 해당 KEY의 할당량 한도 내에서 이용할 수 있습니다.
  • 서비스는 공개된 YouTube 댓글에 한해서만 동작하며, 비공개·삭제된 댓글에는 접근하지 않습니다.
제3조 금지 행위
사용자는 아래 행위를 해서는 안 됩니다. 위반 시 운영자는 사전 통보 없이 서비스 이용을 제한할 수 있습니다.
  • 자동화 스크립트·봇 등을 이용한 과도한 API 호출 및 서비스 부하 유발 행위
  • 서비스를 통해 수집한 데이터의 무단 상업적 재배포 또는 2차 가공 판매
  • 타인의 API KEY를 허가 없이 사용하는 행위
  • YouTube 이용약관, 개인정보보호법 또는 관련 법령을 위반하는 방식으로의 이용
  • 서비스의 정상적인 운영을 방해하는 일체의 행위
제4조 약관의 변경
운영자는 필요한 경우 약관을 변경할 수 있습니다. 변경 시에는 시행일 기준 최소 7일 전에 서비스 내 공지 또는 이메일을 통해 고지합니다. 단, 사용자에게 불리한 변경의 경우 30일 전에 고지합니다. 고지 후 계속 서비스를 이용하는 경우 변경된 약관에 동의한 것으로 간주됩니다.
제5조 서비스 변경 및 중단
운영자는 기술적·운영상 사유로 서비스의 일부 또는 전부를 변경·중단할 수 있습니다. 예측 가능한 경우 사전에 공지하며, 불가피한 경우(서버 장애, 외부 API 정책 변경 등) 사후 고지할 수 있습니다. 운영자의 고의 또는 중대한 과실이 없는 한, 이로 인한 손해에 대해 책임을 지지 않습니다.
제6조 준거법 및 관할
본 약관은 대한민국 법령에 따라 해석되며, 서비스 이용과 관련한 분쟁 발생 시 대한민국 법원을 전속 관할 법원으로 합니다.

Opinion은 사용자의 개인정보를 수집·저장·판매하지 않습니다.
서비스 이용에 회원가입이 필요 없으며, 어떠한 계정 정보도 요구하지 않습니다.

수집하는 정보
① 수집하지 않는 정보
  • 이름, 이메일, 전화번호 등 개인 식별 정보 일체 수집 없음
  • 결제 정보 수집 없음
  • 기기 고유 식별자(IMEI, MAC 등) 수집 없음
  • 위치 정보 수집 없음
② 브라우저 로컬 저장 (기기 내 한정)
아래 정보는 사용자 기기의 브라우저(localStorage)에만 저장되며, 서버로 전송되거나 외부에 공유되지 않습니다.
  • 게스트 분석 횟수 — 일일 무료 할당량 관리 목적 (자정 초기화)
  • 분석 히스토리 — 이전에 분석한 영상 목록 (브라우저 캐시 삭제 시 자동 삭제, 최대 30건)
이 데이터는 오직 해당 기기·브라우저에서만 접근 가능하며, 운영자 또는 제3자는 열람할 수 없습니다.
③ YouTube 댓글 데이터
  • 댓글 데이터는 분석 세션 중에만 메모리에 임시 보관됩니다.
  • 페이지를 새로고침하거나 탭을 닫으면 즉시 소멸됩니다.
  • 수집된 댓글 데이터는 서버 DB에 저장되지 않습니다.
  • CSV·JSON 내보내기 기능을 직접 실행하지 않는 한, 어떠한 파일도 저장되지 않습니다.
④ API KEY 처리

입력하신 YouTube API KEY는 서버 메모리에서 세션 토큰으로 즉시 교환되며, 브라우저나 서버 DB에 평문으로 저장되지 않습니다. 세션 종료 시 토큰은 자동 만료됩니다.

⑤ 서버 접속 로그 (자동 수집)
서비스 서버는 모든 HTTP 요청에 대해 아래 항목을 자동으로 기록합니다. 이는 별도의 코드 없이 서버 인프라(Vercel) 수준에서 수집되는 표준 접속 로그입니다.
  • IP 주소 — 비정상 접근·DDoS 공격 탐지 및 차단, 과도한 API 호출 방지 목적으로만 사용됩니다. 사용자를 개인으로 식별하거나 추적하는 데 활용하지 않습니다.
  • 요청 시각 — 서비스 장애 원인 분석 및 디버깅 목적
  • HTTP 상태코드 및 엔드포인트 — 오류 모니터링 및 서비스 안정성 유지 목적
위 로그는 최대 30일간 보관된 후 자동 삭제됩니다. 로그는 운영·보안 목적 외에 사용되지 않으며, 제3자에게 제공되지 않습니다. 단, 수사기관의 적법한 요청이 있는 경우에는 법령에 따라 제공될 수 있습니다.
⑥ 서비스 이용 통계 (Vercel Analytics)
본 서비스는 Vercel Analytics를 통해 페이지뷰, 방문자 수 등 집계된 통계 데이터를 수집합니다. 이 데이터는 개인을 식별할 수 없는 형태로 집계되며, 서비스 품질 개선 목적으로만 활용됩니다. Vercel의 데이터 처리 방식은 Vercel 개인정보처리방침을 확인하세요.
제3자 서비스
본 서비스는 아래 제3자 서비스를 사용하며, 각 서비스의 개인정보 처리방침이 별도로 적용됩니다.
  • YouTube Data API v3 (Google LLC) — Google 개인정보처리방침 및 YouTube 이용약관 적용
  • Vercel (Vercel Inc.) — 서비스 호스팅 및 Analytics 제공. Vercel 개인정보처리방침 적용

서버 로그 열람·삭제 요청 등 개인정보 관련 문의는 support@inmybackyard.kr로 연락주세요. 접수 후 영업일 기준 7일 이내 답변 드립니다.

YouTube API 서비스 이용 안내
준수 사항
  • 본 서비스는 YouTube API Services Terms of Service(링크)를 준수합니다.
  • YouTube 댓글 데이터는 공개적으로 접근 가능한 정보에 한해서만 수집됩니다.
  • 수집된 데이터는 단순 여론 분석·시각화 목적으로만 사용되며, 상업적으로 재판매하지 않습니다.
  • Google이 API 사용에 관한 데이터 삭제 요청을 한 경우, 해당 요청을 즉시 이행합니다.
API 할당량 및 책임
  • 게스트 모드는 운영자가 제공하는 내장 API KEY를 사용하며, Google의 일일 할당량(10,000 units) 내에서 운영됩니다.
  • 사용자 본인의 API KEY를 사용할 경우, 해당 KEY의 할당량 관리 및 Google 정책 준수 책임은 사용자 본인에게 있습니다.
  • API KEY의 노출·도용으로 인한 할당량 초과 또는 요금 발생에 대해 운영자는 책임을 지지 않습니다.
데이터 사용 제한
