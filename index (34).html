Google 계정(Gmail)이 필요합니다.
API 키는 하루 10,000 units 무료로 제공됩니다.
발급된 키는 절대 타인과 공유하지 마세요.
01
Google Cloud Console 접속
아래 링크를 새 탭에서 열고, Google 계정으로 로그인합니다. 처음 접속 시 서비스 약관 동의 팝업이 뜨면 동의 및 계속하기를 클릭하세요.
console.cloud.google.com 열기
02
새 프로젝트 생성
상단 헤더의 프로젝트 선택 드롭다운을 클릭합니다. 팝업 우측 상단의 새 프로젝트를 클릭하고, 프로젝트 이름을 입력한 뒤 만들기를 누릅니다.
프로젝트 이름은 opinion-analysis 처럼 자유롭게 지정하면 됩니다.
03
YouTube Data API v3 활성화
좌측 메뉴에서 API 및 서비스 → 라이브러리를 클릭합니다. 검색창에 YouTube Data API를 입력하고, 검색 결과에서 YouTube Data API v3를 선택한 뒤 사용 버튼을 클릭합니다.
YouTube Data API v3 바로가기
04
API 키 생성
좌측 메뉴에서 API 및 서비스 → 사용자 인증 정보를 클릭합니다. 상단의 + 사용자 인증 정보 만들기를 클릭하고, API 키를 선택합니다. 잠시 후 키가 생성되면 복사 버튼을 눌러 저장하세요.
사용자 인증 정보 바로가기
05
API 키 제한 설정 (권장)
생성된 키를 클릭해 상세 페이지로 들어갑니다. API 제한 항목에서 키 제한을 선택하고, 드롭다운에서 YouTube Data API v3만 체크한 뒤 저장합니다. 키가 유출되더라도 다른 서비스에서 악용되지 않습니다.
이 단계를 건너뛰어도 동작하지만, 보안을 위해 설정을 권장합니다.
이제 준비 완료!
